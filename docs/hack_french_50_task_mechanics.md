# Hack French: 50 механик заданий

Документ фиксирует набор механик заданий для приложения **Hack French**.

Идея приложения: не «учить французский по урокам», а взламывать язык как систему.  
Каждый кейс — это расследование французского механизма: `en`, `y`, `du`, `lui`, `il faut`, `c’est`, `on`, passé composé, liaison и так далее.

Механики ниже можно комбинировать внутри одного кейса.

---

## Общий цикл кейса

Базовый сценарий прохождения одного кейса:

1. Пользователь видит странность.
2. Собирает улики.
3. Формулирует гипотезу.
4. Разбирает фразу как код.
5. Тренирует механизм на вариациях.
6. Применяет его в бытовой сцене.
7. Создаёт свою фразу.
8. Получает мини-озарение.
9. Идёт ловить механизм в реальной жизни.

---

# 1. Найди странность

## Суть

Пользователь видит французскую фразу и должен заметить, что в ней устроено не так, как ожидал бы русский мозг.

## Зачем нужна

Эта механика включает режим наблюдателя.  
Пользователь перестаёт просто переводить слова и начинает замечать внутреннюю логику языка.

## Пример

```fr
J’ai froid.
```

Вопрос:

> Что здесь странного для русского мозга?

Варианты:

- французы говорят «я имею холод»;
- французы используют будущее время;
- французы опускают подлежащее;
- французы говорят через местоимение `y`.

Правильная идея:

> Во французском состояние «мне холодно» выражается через `avoir`.

## Подходит для кейсов

- `j’ai froid`;
- `j’ai faim`;
- `j’ai besoin de`;
- `j’ai envie de`;
- артикли;
- `en`;
- `y`;
- `lui`;
- `il faut`;
- `on`;
- разговорные сокращения.

# 2. Выбери правильную гипотезу

## Суть

Пользователь видит несколько примеров и выбирает гипотезу, которая лучше всего объясняет механизм.

## Зачем нужна

Это превращает грамматику в маленькое научное расследование.  
Пользователь не получает правило сразу, а как будто сам выводит его по уликам.

## Пример

```fr
J’en veux.
J’en ai besoin.
J’en prends deux.
```

Вопрос:

> Что, скорее всего, делает `en`?

Варианты:

- заменяет кусок с `de`;
- показывает будущее;
- делает фразу вежливой;
- означает человека.

Ответ:

> `en` часто заменяет кусок с `de`: `de ça`, `du café`, `des croissants`.

## Подходит для кейсов

- `en`;
- `y`;
- `lui / leur`;
- артикли;
- `depuis / pendant / il y a`;
- passé composé vs imparfait;
- `c’est / il est`.

# 3. Разбор как код

## Суть

Фраза разбирается на блоки, почти как строка кода.

## Зачем нужна

Пользователь видит не «магическую французскую фразу», а конструкцию из понятных деталей.

## Пример

```fr
Je lui ai parlé.
```

Разбор:

```text
Je = я
lui = ему / ей
ai parlé = поговорил
```

Или визуально:

```text
[Je] [lui] [ai parlé]
 кто  кому  что сделал
```

## Подходит для кейсов

- местоимения;
- passé composé;
- `il faut`;
- `je voudrais`;
- `j’ai besoin de`;
- `y`;
- `en`;
- вопросы;
- отрицание.

# 4. Собери фразу из блоков

## Суть

Пользователь получает набор блоков и должен собрать правильную французскую фразу.

## Зачем нужна

Это тренирует порядок слов без необходимости сразу писать фразу с нуля.

## Пример

Задание:

> Собери: «Я ему сказал».

Блоки:

```text
Je / lui / ai / dit
```

Ответ:

```fr
Je lui ai dit.
```

## Подходит для кейсов

- местоимения перед глаголом;
- passé composé;
- `je te le donne`;
- `je voudrais`;
- `il faut`;
- futur proche;
- вопросы через `est-ce que`.

# 5. Почини русский перевод

## Суть

Пользователь видит буквальный кривой перевод французской фразы и должен восстановить нормальный русский смысл.

## Зачем нужна

Механика помогает не бояться французской логики и отделять буквальную структуру от реального смысла.

## Пример

```fr
J’ai envie de dormir.
```

Буквально:

> Я имею желание спать.

Задание:

> Как это нормально сказать по-русски?

Ответ:

> Мне хочется спать.

## Подходит для кейсов

- `avoir envie de`;
- `avoir besoin de`;
- `avoir mal à`;
- `il y a`;
- `il faut`;
- `je vais chercher`;
- `ça va`;
- `en`;
- `y`.

# 6. Переведи не словами, а механизмом

## Суть

Пользователь получает русскую фразу, но должен выбрать не буквальный перевод, а правильный французский механизм.

## Зачем нужна

Обычный перевод провоцирует кальку.  
Эта механика учит выбирать французскую конструкцию.

## Пример

Задание:

> Скажи по-французски: «Мне холодно».

Варианты:

```fr
Je suis froid.
J’ai froid.
Il fait froid moi.
Moi froid.
```

Ответ:

```fr
J’ai froid.
```

## Подходит для кейсов

- `avoir` для состояний;
- возраст через `avoir`;
- `j’ai besoin de`;
- `j’ai envie de`;
- `il faut`;
- `je voudrais`;
- артикли;
- `chez`.

# 7. Найди лишнюю фразу

## Суть

Пользователь видит несколько фраз. Одна из них работает по другому механизму.

## Зачем нужна

Механика тренирует различение похожих конструкций.

## Пример

```fr
Je lui parle.
Je lui écris.
Je lui donne un livre.
Je le vois.
```

Вопрос:

> Какая фраза выбивается?

Ответ:

```fr
Je le vois.
```

Почему:

> `le` — прямой объект, а `lui` — «кому».

## Подходит для кейсов

- `le / la / les` vs `lui / leur`;
- `savoir` vs `connaître`;
- `voir` vs `regarder`;
- `écouter` vs `entendre`;
- `un / le / du`;
- passé composé vs imparfait.

# 8. Мини-расследование по уликам

## Суть

Пользователь получает несколько примеров и должен сам вывести правило.

## Зачем нужна

Это центральная механика Hack French.  
Правило не навязывается сверху, а рождается из наблюдения.

## Пример

```fr
Je vais à l’école → J’y vais.
Je pense à ça → J’y pense.
Je suis à Toulouse → J’y suis.
```

Вопрос:

> Что заменяет `y`?

Ответ:

> `y` часто заменяет место или идею, связанные с `à`.

## Подходит для кейсов

- `y`;
- `en`;
- артикли;
- местоимения;
- `depuis / pendant / il y a`;
- `c’est / il est`;
- `on`.

# 9. Выбери фразу для сцены

## Суть

Пользователь получает бытовую ситуацию и выбирает естественную французскую фразу.

## Зачем нужна

Механика связывает грамматику с жизнью: булочная, школа, врач, администрация.

## Пример

Сцена:

> Ты в булочной. Продавец спрашивает, сколько круассанов ты хочешь. Скажи: «Я возьму два».

Варианты:

```fr
Je prends deux.
J’en prends deux.
Je les prends deux.
Je prends en deux.
```

Ответ:

```fr
J’en prends deux.
```

Почему:

> `en` заменяет объект, о котором уже говорят: croissants.

## Подходит для кейсов

- `en`;
- `je voudrais`;
- `il faut`;
- `j’ai besoin de`;
- `chez`;
- `prendre rendez-vous`;
- фразы-спасатели.

# 10. Мутация фразы

## Суть

Пользователь берёт один каркас и меняет в нём детали.

## Зачем нужна

Так формируется чувство паттерна.  
Пользователь не учит одну фразу, а понимает, как её адаптировать.

## Пример

Каркас:

```fr
Je voudrais prendre rendez-vous.
```

Задание:

> Сделай фразу: «Я хотел бы перенести встречу».

Ответ:

```fr
Je voudrais déplacer mon rendez-vous.
```

Другие мутации:

```fr
Je voudrais annuler mon rendez-vous.
Je voudrais confirmer mon rendez-vous.
Je voudrais prendre rendez-vous avec le médecin.
```

## Подходит для кейсов

- `je voudrais`;
- `j’ai besoin de`;
- `il faut`;
- `je vais devoir`;
- futur proche;
- бытовые сцены;
- телефонные разговоры.

# 11. Переключи регистр

## Суть

Пользователь выражает одну мысль в разных уровнях вежливости или формальности.

## Зачем нужна

Французский — это не только грамматика, но и социальная дистанция.  
Механика помогает не звучать грубо.

## Пример

Идея:

> Хочу кофе.

Прямо:

```fr
Je veux un café.
```

Нейтрально-вежливо:

```fr
Je voudrais un café.
```

Очень мягко:

```fr
J’aimerais prendre un café, s’il vous plaît.
```

## Подходит для кейсов

- `je veux` vs `je voudrais`;
- conditionnel;
- просьбы;
- `tu / vous`;
- телефон;
- администрация;
- кафе;
- школа.

# 13. Разверни фразу

## Суть

Обратная механика: пользователь видит короткую фразу и должен понять, что спрятано в маленьком слове.

## Зачем нужна

Она развивает понимание настоящей речи, где французы часто не повторяют объект целиком.

## Пример

```fr
J’en ai besoin.
```

Вопрос:

> Что может быть спрятано в `en`?

Варианты:

```text
de ce document
à l’école
mon fils
le café
```

Ответ:

```text
de ce document
```

## Подходит для кейсов

- `en`;
- `y`;
- объектные местоимения;
- контекстные диалоги;
- listening comprehension.

# 14. Выбери правильную камеру

## Суть

Механика для артиклей: пользователь выбирает, как французский «видит» объект.

## Зачем нужна

Артикли трудно учить как таблицу.  
Их проще почувствовать как настройку камеры:

- новый объект;
- известный объект;
- категория;
- порция;
- неопределённое количество.

## Пример

Сцена:

> Ты впервые вводишь объект в разговор: «У меня есть вопрос».

Варианты:

```fr
J’ai question.
J’ai une question.
J’ai la question.
J’ai de la question.
```

Ответ:

```fr
J’ai une question.
```

## Подходит для кейсов

- `un / une`;
- `le / la / les`;
- `du / de la / des`;
- `pas de`;
- еда;
- абстрактные понятия.

# 15. Камера: категория или порция

## Суть

Пользователь определяет, где речь о категории в целом, а где о порции или неопределённом количестве.

## Зачем нужна

Это один из ключевых инсайтов для артиклей.

## Пример

Разложить по корзинам:

```fr
J’aime le fromage.
Je veux du fromage.
Je mange du pain.
Je déteste les tomates.
```

Корзины:

- категория;
- порция / некоторое количество.

Ответ:

Категория:

```fr
J’aime le fromage.
Je déteste les tomates.
```

Порция / количество:

```fr
Je veux du fromage.
Je mange du pain.
```

## Подходит для кейсов

- `j’aime le...`;
- `je veux du...`;
- еда;
- покупки;
- кафе;
- булочная.

# 16. Исправь кальку

## Суть

Пользователь видит типичную ошибку русскоязычного и исправляет её.

## Зачем нужна

Механика показывает, какой русский шаблон мешает говорить по-французски.

## Пример

Ошибка:

```fr
Je suis 35 ans.
```

Исправление:

```fr
J’ai 35 ans.
```

Другие примеры:

```fr
J’ai question. → J’ai une question.
Je suis froid. → J’ai froid.
Je vais à médecin. → Je vais chez le médecin.
Je parle lui. → Je lui parle.
```

## Подходит для кейсов

- возраст;
- `avoir` для состояний;
- артикли;
- `chez`;
- местоимения;
- порядок слов;
- ложные друзья.

# 17. Объясни ошибку

## Суть

Пользователь не просто исправляет ошибку, а выбирает объяснение, почему это ошибка.

## Зачем нужна

Так формируется осознанное понимание механизма, а не механическое запоминание.

## Пример

Ошибка:

```fr
Je suis froid.
```

Вопрос:

> Почему это ошибка?

Варианты:

- потому что состояние «мне холодно» во французском выражается через `avoir`;
- потому что `froid` нельзя использовать с людьми;
- потому что нужен passé composé;
- потому что надо использовать `y`.

Ответ:

> Состояние «мне холодно» выражается через `avoir`: `j’ai froid`.

## Подходит для кейсов

- все типичные ошибки;
- ловушки русскоязычного мозга;
- артикли;
- местоимения;
- времена;
- предлоги.

# 19. Сравни две фразы

## Суть

Пользователь видит две похожие фразы и объясняет, чем они отличаются.

## Зачем нужна

Механика развивает тонкое различение близких конструкций.


> Почему здесь разные глаголы?

- `an / année`;
- `c’est / il est`;
- passé composé / imparfait;
- `je veux / je voudrais`.

- `ça va`;

# 21. Выбери естественную фразу

## Суть

Пользователь выбирает не просто грамматически правильную, а наиболее естественную фразу для ситуации.

## Зачем нужна

Цель Hack French — не только правильность, но и естественность.

## Пример

Сцена:

> Ты хочешь спросить: «Можно войти?»

Варианты:

```fr
Puis-je entrer ?
Est-ce que je peux entrer ?
Je peux entrer ?
Entrer je peux ?
```

Возможный разбор:

- `Est-ce que je peux entrer ?` — нейтрально;
- `Je peux entrer ?` — разговорно;
- `Puis-je entrer ?` — формально;
- `Entrer je peux ?` — ошибка.

## Подходит для кейсов

- вопросы;
- просьбы;
- регистр;
- `je voudrais`;
- `tu / vous`;
- бытовые сцены.

# 22. Пересобери из формального в живое

```fr

```

- `ça va`;
- `du coup`;
- `en fait`.

# 24. Выбери правильный микросигнал

## Суть

Задания:

1. понять смысл;
2. разобрать как код;
3. найти механизм `aller chercher`;
4. сделать 3 вариации;
5. произнести вслух;
6. попробовать использовать в жизни.

Вариации:

```fr
Je vais chercher mon fils à la crèche.
Je vais chercher le pain.
Je vais chercher ma commande.
```

## Подходит для кейсов

- персональные сценарии;
- crèche;
- école;
- boulangerie;
- pharmacie;
- rendez-vous;
- повседневная речь.

12. Босс-файт

# Как комбинировать механики внутри одного кейса

Пример кейса: `en`.

## Возможный сценарий

1. **Найди странность**

```fr
J’en veux.
```

2. **Улики**

```fr
J’en veux.
J’en ai besoin.
J’en prends deux.
J’en parle.
```

3. **Выбери гипотезу**

> Что делает `en`?

4. **Разбор как код**

```fr
Je veux du café.
J’en veux.
```

5. **Сверни фразу**

```fr
Je prends deux croissants. → J’en prends deux.
```

6. **Разверни фразу**

```fr
J’en ai besoin. → J’ai besoin de ce document.
```

7. **Сцена**

> Ты в булочной. Скажи: «Я возьму два».

8. **Своя фраза**

> Напиши 3 фразы из своей жизни с `en`.

9. **Озарение**

> `En` — это французский способ не повторять «этого / из этого / немного / их».

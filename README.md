# Hack French 🕵️‍♂️🇫🇷

> Не зубри французский. Взламывай, как он работает.

**Hack French** — это игра-лаборатория, где французский язык учат как детектив раскрывает дела.
Никаких бесконечных таблиц и зубрёжки: вы замечаете странную французскую фразу, собираете улики,
вычисляете её внутренний механизм, разбираете фразу «как код», применяете в живой бытовой сцене —
и уходите с коротким **озарением**, которое остаётся в голове.

Приложение на русском языке и рассчитано на русскоговорящих, которые хотят **понять** французский,
а не просто запомнить.

## Как это работает

Каждое **дело** — это маленькое расследование одного механизма французского:

1. 🔍 **Странность.** Видите фразу, которая «не должна так работать».
2. 🧩 **Улики и гипотеза.** Собираете подсказки и выдвигаете версию.
3. 💻 **Вскрытие.** Разбираете фразу по кусочкам, как строчку кода.
4. 🎭 **Сцена.** Применяете механизм в реальной ситуации — в булочной, у врача, в кафе.
5. 💡 **Озарение.** Формулируете правило одним ясным предложением — и оно ваше.

## Что внутри

- 🗂️ **108 дел в 19 разделах** — от артиклей и отрицания до разговорных ловушек и боссов.
- 🎮 **24 разные механики** вместо однообразных тестов: собери фразу из блоков, исправь кальку,
  поймай ложного друга-переводчика, пройди диалог-квест, найди баг во фразе в «French Debug Mode»
  и другие.
- 🔁 **Летучка** — умное повторение. Раскрытые дела со временем «остывают» и возвращаются по
  графику (1 → 3 → 7 → 21 → 60 дней), чтобы механизмы закреплялись, а не забывались. Есть карта
  крепости механизмов и серии верных ответов.
- 🎯 **Журнал миссий** — задания «поймай механизм в реальной жизни» и ваши собственные фразы,
  составленные под себя. Здесь язык связывается с вашей жизнью.
- 🏆 **Коллекция озарений** — все правила, которые вы вывели сами, в одном месте.
- 🕵️ **Атмосфера детективного досье** — тёмная тема, шрифты-улики и аккуратная типографика
  вместо безликого интерфейса.

## Прогресс и приватность

- Всё работает **в браузере, офлайн, без аккаунта и без сервера** — никакой регистрации.
- Прогресс хранится локально в вашем браузере.
- Переехать на другое устройство можно одним файлом: в **Настройках** прогресс выгружается в
  `.json` и загружается на новом устройстве.

## Технологии

React 19 · TypeScript · Vite · MUI · React Router — пакетный менеджер и рантайм **[bun](https://bun.sh)**.

## Команды

```bash
bun install      # установить зависимости
bun run dev      # дев-сервер (http://localhost:5173)
bun run build    # прод-сборка (tsc -b + vite build)
bun run preview  # предпросмотр прод-сборки
bun run lint     # ESLint
bun run format   # Prettier
```
